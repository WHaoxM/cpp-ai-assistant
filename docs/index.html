概述

C++ AI 助手 API 提供了强大的自然语言处理能力，支持多种编程语言的代码生成、题目分析和知识问答。本 API 兼容 OpenAI 格式，方便开发者快速集成。

基础信息

  • Base URL: https://api.example.com/v1
  • 协议: HTTPS
  • 数据格式: JSON
  • 字符编码: UTF-8

认证

所有 API 请求都需要在 HTTP Header 中包含 API Key 进行身份验证。

请求头

Header Authorization: Bearer <YOUR_API_KEY>

获取 API Key

1. 注册账号并登录

2. 进入控制台创建 API Key

3. 复制 API Key 并妥善保管

聊天补全

创建一个聊天补全请求，返回 AI 的响应。

端点

POST /chat/completions

请求示例

curl --request POST \
  --url https://api.example.com/v1/chat/completions \
  --header 'Authorization: Bearer <YOUR_API_KEY>' \
  --header 'Content-Type: application/json' \
  --data '{
  "model": "qwen3-max",
  "messages": [
    {
      "role": "user",
      "content": "解释一下 C++ 中的指针概念"
    }
  ],
  "stream": false,
  "max_tokens": 512,
  "temperature": 0.7,
  "top_p": 0.7
}'

参数说明

参数名 类型 必需 说明
model string 是 使用的模型 ID，如：qwen3-max, gpt-4, claude-3
messages array 是 消息列表，包含对话历史
stream boolean 否 是否使用流式输出，默认 false
max_tokens integer 否 最大生成 token 数，默认 512
temperature float 否 采样温度，范围 0-2，默认 0.7
top_p float 否 核采样概率，范围 0-1，默认 0.7
top_k integer 否 保留前 k 个候选，默认 50
frequency_penalty float 否 频率惩罚，范围 -2.0 到 2.0，默认 0.5
n integer 否 生成多少个结果，默认 1

响应格式

成功响应

{
  "id": "chatcmpl-123456789",
  "object": "chat.completion",
  "created": 1699012345,
  "model": "qwen3-max",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "在 C++ 中，指针是一个变量，其值为另一个变量的地址..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 50,
    "completion_tokens": 100,
    "total_tokens": 150
  }
}

字段说明

  • id: 响应的唯一标识符
  • object: 对象类型，固定为 "chat.completion"
  • created: 创建时间戳
  • model: 使用的模型
  • choices: 生成的结果列表
  • usage: token 使用统计

示例代码

Python 示例

import requests

url = "https://api.example.com/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "model": "qwen3-max",
    "messages": [
        {
            "role": "user",
            "content": "解释一下 C++ 中的指针概念"
        }
    ],
    "temperature": 0.7
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

JavaScript 示例

const response = await fetch('https://api.example.com/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'qwen3-max',
    messages: [
      {
        role: 'user',
        content: '解释一下 C++ 中的指针概念'
      }
    ],
    temperature: 0.7
  })
});

const data = await response.json();
console.log(data);

错误码

HTTP 状态码 错误码 说明
401 UNAUTHORIZED API Key 无效或已过期
429 TOO_MANY_REQUESTS 请求过于频繁，请降低请求频率
500 INTERNAL_ERROR 服务器内部错误，请稍后重试
400 INVALID_REQUEST 请求参数错误
